# [Baekjoon] 2720번: 세탁소 사장 동혁 (C언어)

## 1. 문제 설명
거스름돈 액수가 주어졌을 때, 쿼터(25¢), 다임(10¢), 니켈(5¢), 페니(1¢)의 개수를 최소화하여 거슬러 주는 문제입니다.

## 2. 해결 전략: 그리디(Greedy) 알고리즘
동전의 개수를 최소로 하려면 **가장 가치가 높은 동전부터** 최대한 많이 사용해야 합니다. 
이 문제처럼 큰 단위가 작은 단위의 배수일 경우, 그리디 알고리즘으로 최적의 해를 구할 수 있습니다.



## 3. 핵심 로직
1. **입력:** 테스트 케이스 개수 $T$를 입력받고, $T$번 반복합니다.
2. **나눗셈(/):** 거스름돈을 각 동전의 단위로 나눈 **몫**이 해당 동전의 개수가 됩니다.
3. **나머지(%):** 나눈 후의 **나머지**를 다음 동전 단위의 거스름돈으로 갱신합니다.
4. **순서:** 25 → 10 → 5 → 1 순서로 계산합니다.
