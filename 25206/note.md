# 📝 [오답노트] 백준 25206번: 너의 평점은

## 1. 🚨 치명적인 실수 (Critical Mistakes)

### ① 변수 초기화 누락 (Uninitialized Variables)
* **문제 상황**:
    ```c
    double sum_score, sum_credit; 
    // ...
    sum_score += n * grade; // 쓰레기 값 + 계산값 = ?
    ```
* **원인 분석**: 
    * C언어에서 `main` 함수 내부의 지역 변수는 자동으로 0으로 초기화되지 않습니다.
    * 메모리에 남아있던 임의의 값(Garbage Value)에 숫자를 계속 더하게 되어, 결과가 `4.2341...` 처럼 엉뚱하게 나오거나 `NaN`(Not a Number)이 뜰 수 있습니다.
* **해결**: 누적 연산(`+=`)을 사용하는 변수는 **반드시 선언과 동시에 0으로 초기화**해야 합니다.
    ```c
    double sum_score = 0.0;
    double sum_credit = 0.0;
    ```

### ② 대입 연산자 오용 (Assignment inside Condition)
* **문제 상황**:
    ```c
    else if(score[0] = 'B') // 'B'를 넣어버림!
    ```
* **원인 분석**: 
    * `==` (비교) 대신 `=` (대입)을 사용했습니다.
    * 이 코드는 `score[0]`의 값을 강제로 'B'로 바꾼 뒤, 그 결과값('B'의 아스키 코드값, 0이 아님)을 참(True)으로 인식합니다.
    * 결과적으로 **A학점이든 C학점이든 무조건 B학점(3.0)으로 둔갑**하여 계산됩니다.
* **해결**: 비교 연산자 `==`를 정확히 사용해야 합니다.
    ```c
    else if(score[0] == 'B')
    ```

## 2. 💡 자료형과 문자열 처리 (Data Types & Strings)

### ① 문자(char) vs 문자열(char array)
* **문제점**: `char score;`로 선언하면 `'A+'` 같은 두 글자를 저장할 수 없습니다. (마지막 글자만 남거나 오버플로우 발생)
* **해결**: 문자열을 저장하려면 넉넉한 크기의 배열이 필요합니다.
    * `char score[5];` (등급용)
    * `char name[51];` (과목명용)

### ② 문자열 비교 방법
* **잘못된 접근**: `if (score == "A+")` 
    * C언어에서 문자열은 `==` 연산자로 내용을 비교할 수 없습니다. (주소값 비교가 됨)
* **해결 방법 1 (정석)**: `strcmp(score, "A+") == 0` 함수 사용 (`string.h` 필요).
* **해결 방법 2 (최적화)**: 문자 하나씩 뜯어서 비교하기 (아래 로직 참조).

## 3. 🚀 로직 최적화 (Code Refactoring)

모든 경우의 수(`A+`, `A0`, `B+`...)를 `if-else`로 나열하는 것은 코드가 길어지고 오타 확률을 높입니다. **규칙성**을 이용해 코드를 줄였습니다.

### 알고리즘 설계
1.  **Pass 처리**: `score[0]`이 'P'면 `continue`로 즉시 건너뜀.
2.  **Fail 처리**: `score[0]`이 'F'면 0.0점.
3.  **점수 계산 (A~D)**:
    * **기본 점수**: 
        * 'A' → 4.0
        * 'B' → 3.0
        * 'C' → 2.0
        * 'D' → 1.0
    * **추가 점수**: 
        * `score[1]`(두 번째 글자)이 `+`라면 **0.5점 추가**.
        * `0`이라면 그대로 둠.

### 적용 코드 (핵심 부분)
```c
if (score[0] == 'A') grade = 4.0;
else if (score[0] == 'B') grade = 3.0;
else if (score[0] == 'C') grade = 2.0;
else grade = 1.0;

if (score[1] == '+') grade += 0.5; // '+'가 있으면 0.5점 Up
```
## 4. 📌 결론 (Takeaway)

1. **초기화 습관**: 
   - 합계를 구하는 변수는 무조건 `0` (또는 `0.0`)으로 초기화하자.
2. **연산자 확인**: 
   - `if`문 안에는 대입(`=`)이 아니라 비교(`==`)가 들어가야 한다.
3. **문자열 분해**: 
   - 문자열 전체를 비교하는 것보다, 특징적인 문자(첫 글자, 두 번째 글자)만 확인하는 것이 더 효율적일 때가 있다.
