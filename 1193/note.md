# [백준 1193] 분수찾기 - C언어

## 1. 문제 핵심 요약
이 문제는 무한히 큰 배열을 지그재그 방향으로 돌며 $X$번째 분수를 찾는 문제입니다. 단순히 배열을 만드는 것이 아니라, **수학적 규칙(군수열)**을 파악하여 $O(N)$ 미만의 효율로 풀어야 합니다.



## 2. 로직 설계
### ① $X$가 속한 라인($n$) 찾기
각 라인은 라인 번호만큼의 칸을 가집니다. (1라인 1개, 2라인 2개...)
입력값 $X$에서 1, 2, 3...을 차례로 빼며 $X$가 몇 번째 라인에 있는지, 그리고 그 라인에서 몇 번째 위치에 있는지를 구합니다.

```c
int n = 1;
while (X > n) {
    X -= n;
    n++;
}
// 결과: n은 라인 번호, X는 해당 라인 내에서의 순서
```



### ② 지그재그 방향 처리 (짝수/홀수 라인 구분)

이 문제의 핵심은 **라인 번호($n$)**에 따라 분수가 배치되는 방향이 다르다는 점입니다.



| 라인 ($n$) | 진행 방향 | 분자 (Numerator) | 분모 (Denominator) | 비고 |
| :--- | :---: | :--- | :--- | :--- |
| **짝수 라인** | **↙** (내려감) | $X$ | $n - X + 1$ | 분자 $1 \rightarrow n$ 증가 |
| **홀수 라인** | **↗** (올라감) | $n - X + 1$ | $X$ | 분자 $n \rightarrow 1$ 감소 |

#### 로직 상세 설명
1. **짝수 라인 (2, 4, 6...)**: 대각선이 왼쪽 아래로 내려갑니다. 라인 내 순서($X$)가 커질수록 분자도 같이 1, 2, 3... 순으로 커집니다.
2. **홀수 라인 (1, 3, 5...)**: 대각선이 오른쪽 위로 올라갑니다. 라인 내 순서($X$)가 커질수록 분자는 $n$부터 시작해 1씩 작아집니다.

#### C언어 적용 예시
```c
if (n % 2 == 0) {
    // 짝수일 때: 분자 = X, 분모 = n - X + 1
    printf("%d/%d\n", X, n - X + 1);
} else {
    // 홀수일 때: 분자 = n - X + 1, 분모 = X
    printf("%d/%d\n", n - X + 1, X);
