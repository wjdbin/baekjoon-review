# 📝 오답노트: 백준 2566번 (핵심 요약)

### 1. 틀린 이유 (Cause)
* `max`는 `arr[0][0]`으로 초기화했으나, 위치 변수 `m_i`, `m_j`는 초기화하지 않음.
* 최댓값이 **첫 번째 칸(1행 1열)**에 있으면 `if(값 > max)` 조건이 거짓이 되어, 위치 변수에 값이 할당되지 않음 (쓰레기값 출력).

### 2. 해결 (Solution)
위치 변수도 반드시 기본값(1행 1열)으로 초기화해야 함.

```c
// [수정 전] int m_i, m_j; 
int m_i = 1, m_j = 1; // [수정 후] ✅ (1,1)로 초기화
