# [백준 2292] 벌집 - C언어 풀이

## 1. 문제 분석 및 규칙 찾기
벌집의 숫자가 늘어나는 규칙을 살펴보면, 중심 '1'을 기준으로 한 층씩 멀어질 때마다 각 층의 마지막 숫자(최댓값)가 **6의 배수**만큼 늘어나는 것을 알 수 있습니다.

| 층 (이동 거리) | 숫자 범위 | 해당 층의 최댓값 (계산식) |
| :---: | :--- | :--- |
| **1층** | 1 | 1 |
| **2층** | 2 ~ 7 | 1 + (6 * 1) = 7 |
| **3층** | 8 ~ 19 | 7 + (6 * 2) = 19 |
| **4층** | 20 ~ 37 | 19 + (6 * 3) = 37 |

## 2. 풀이 전략
- **변수 설정**: 
  - `range_max`: 해당 층의 가장 큰 숫자 (초기값 1)
  - `count`: 벌집의 층수 (결과값, 초기값 1)
- **반복문**:
  - 사용자가 입력한 숫자 `n`이 현재 층의 최댓값(`range_max`)보다 크다면, 아직 목표 숫자에 도달하지 못한 것입니다.
  - 다음 층으로 넘어가기 위해 `range_max`에 `6 * count`를 더해주고, `count`를 1 증가시킵니다.
  - `n <= range_max`가 되는 순간의 `count`가 정답이 됩니다.
