# [백준 2903] 중앙 이동 알고리즘 - 문제 풀이 정리

### 1. 문제 분석
이 문제는 각 단계($N$)마다 사각형의 변에 놓이는 점의 개수가 어떻게 변하는지 파악하는 것이 핵심입니다.

- **규칙**: 한 변의 점의 개수는 이전 단계의 변의 점들 사이에 새로운 점이 추가되면서 늘어납니다.
- **공식**: 
  - 한 변의 점 개수 = $2^N + 1$
  - 전체 점의 개수 = $(2^N + 1)^2$

| 단계 ($N$) | 한 변의 점 개수 | 전체 점의 개수 (결과) |
|:---:|:---:|:---:|
| 0 | $2^0 + 1 = 2$ | $2^2 = 4$ |
| 1 | $2^1 + 1 = 3$ | $3^2 = 9$ |
| 2 | $2^2 + 1 = 5$ | $5^2 = 25$ |
| 5 | $2^5 + 1 = 33$ | $33^2 = 1089$ |
