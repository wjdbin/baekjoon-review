# 백준 3003번: C언어 핵심 요약

## ♟️ 문제 핵심

* **기준:** 킹 1, 퀸 1, 룩 2, 비숍 2, 나이트 2, 폰 8
* **입력 (a~f):** 가지고 있는 개수
* **출력:** `(기준 개수) - (가진 개수)`를 순서대로 출력

## 💻 C언어 개념

### 1. `#include <stdio.h>`
* **St**an**d**ard **I**nput **O**utput (표준 입출력)
* `scanf` (입력)와 `printf` (출력) 함수를 쓰기 위해 필요합니다.

### 2. `int a, b, c, d, e, f;`
* `int` (정수) 값을 저장할 6개의 변수(저장 공간)를 선언합니다.

### 3. `scanf("%d %d %d %d %d %d", &a, &b, ...);`
* 키보드로 6개의 정수(`%d`)를 입력받습니다.
* **`&` (주소 연산자):** `scanf`는 값을 *저장할 위치(주소)*를 알아야 하므로 `&`를 **반드시** 붙입니다.

### 4. `printf("%d %d %d %d %d %d", 1 - a, 1 - b, ...);`
* 6개의 계산된 정수(`%d`) 값을 모니터에 출력합니다.
* `1 - a`처럼 **계산된 결과(값)**를 전달합니다. (`&`를 붙이지 않습니다.)
