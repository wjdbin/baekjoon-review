# [BOJ] 2745번: 진법 변환 (C언어)

## 1. 문제 정보
- **문제 번호:** 2745
- **문제 이름:** 진법 변환
- **문제 링크:** [https://www.acmicpc.net/problem/2745](https://www.acmicpc.net/problem/2745)
- **난이도:** 브론즈 II

---

## 2. 문제 이해
B진법으로 표기된 수 `N`과 진법 `B`가 주어지면, 이를 **10진법**으로 변환하여 출력하는 문제이다.
10을 넘어가는 진법(11진법~36진법)의 경우 숫자로 표기할 수 없으므로 알파벳 대문자 `A`부터 `Z`를 사용한다.
* `A`: 10, `B`: 11, ..., `Z`: 35

---

## 3. 핵심 풀이 전략
### 1) 문자(char)를 숫자(int)로 변환하기 (ASCII 코드 활용)
입력받은 `N`은 문자열이므로, 각 자리의 문자가 실제 어떤 값을 가지는지 변환해야 한다.
- **'0' ~ '9'인 경우:** 문자에서 `'0'`(아스키 48)을 빼면 정수 0~9가 된다.
  - `num = c - '0';`
- **'A' ~ 'Z'인 경우:** 문자에서 `'A'`(아스키 65)를 빼면 0~25가 되는데, 문제에서 A는 10부터 시작하므로 **10을 더해준다.**
  - `num = c - 'A' + 10;`

### 2) 진법 변환 계산 (Horner's Method)
가장 앞자리(왼쪽)부터 문자를 읽으며 내려가는 방식이 효율적이다.
- 이전 결과값에 진법 `B`를 곱하고, 현재 자리의 숫자를 더한다.
- 공식: `result = result * B + current_digit`

---

## 4. 소스 코드 (C)

```c
#include <stdio.h>
#include <string.h>

int main() {
    char N[1000]; // B진법 수 N을 저장할 문자열
    int B;        // 진법 B
    int result = 0; // 10진법으로 변환한 결과

    // 입력 받기
    scanf("%s %d", N, &B);

    int len = strlen(N); // 문자열 길이 확인

    // 앞에서부터 한 글자씩 읽으며 계산
    for (int i = 0; i < len; i++) {
        char c = N[i];
        int num;

        // 1. 문자를 숫자로 변환
        if (c >= '0' && c <= '9') {
            num = c - '0';
        } else {
            num = c - 'A' + 10;
        }

        // 2. 진법 변환 (누적)
        // 기존 값에 B를 곱해서 자릿수를 올리고 현재 값을 더함
        result = result * B + num;
    }

    printf("%d\n", result);

    return 0;
}
