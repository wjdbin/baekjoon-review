# [Baekjoon] 11005번: 진법 변환 2

## 📝 문제 개요
10진법 수 `N`을 입력받아 `B`진법으로 변환하여 출력하는 문제.
(단, 10진법을 넘어가는 10~35의 숫자는 알파벳 대문자 A~Z로 표기한다.)

## 💡 핵심 로직
1. **나눗셈 연산**: 10진수 `N`을 `B`로 나누면서, 그 **나머지(Remainder)**가 해당 자릿수의 값이 된다.
2. **갱신**: `N`을 `B`로 나눈 **몫(Quotient)**으로 갱신하며 `N`이 0이 될 때까지 반복한다.
3. **아스키 코드 매핑**:
   - 나머지 `0 ~ 9`: `'0' + 나머지`
   - 나머지 `10 ~ 35`: `'A' + (나머지 - 10)`
4. **역순 출력**: 나눗셈 방식으로 구한 자릿수는 **낮은 자리(일의 자리)**부터 구해지므로, 저장된 결과를 **거꾸로 뒤집어서** 출력해야 한다.
